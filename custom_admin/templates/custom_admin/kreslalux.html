Запускати локально на своєму ПК

Сайт kreslalux.ua блокує запити з сервера (Cloudflare 403). Команда виконується локально — скрейп, завантаження картинок на R2 і запис у БД відбуваються на вашому ПК.

Всі товари до 40 000 грн:

python manage.py import_kreslalux_chairs

Тест (3 товари, без змін):

python manage.py import_kreslalux_chairs --dry-run --limit 3

Якщо локальна БД відрізняється від прод:

python manage.py import_kreslalux_chairs --database-url postgres://...

Оновлення цін

Також запускати локально

Оновлює ціни наявних товарів. Наявні артикули порівнюються з каталогом kreslalux.ua. Нові товари також додаються.

Оновити ціни (тільки існуючі):

python manage.py import_kreslalux_chairs

Команда сама визначає що нове, а що вже є — оновлює ціну при розбіжності.
