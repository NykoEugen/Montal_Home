Повний імпорт (2 сторінки каталогу):

python manage.py import_divanoff

Тест (без змін у БД):

python manage.py import_divanoff --dry-run

Обмежений тест (5 URL):

python manage.py import_divanoff --dry-run --limit 5

З іншою категорією ткани (3=кат.I, 4=кат.II ... 9=кат.VII):

python manage.py import_divanoff --price-col 3

На продакшн БД:

python manage.py import_divanoff --database-url postgres://...
