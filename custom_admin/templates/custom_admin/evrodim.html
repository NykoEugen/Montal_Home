Запускати локально на своєму ПК

Обходить всі сторінки каталогу (8 сторінок), фільтрує товари «Стіл*» з виробником Generic, завантажує картинки та записує в БД.

Всі столи Generic:

python manage.py import_evrodim_tables

Тест (без змін у БД):

python manage.py import_evrodim_tables --dry-run

Обмежений тест (5 URL з каталогу):

python manage.py import_evrodim_tables --dry-run --limit 5

Інша підкатегорія:

python manage.py import_evrodim_tables --subcategory your-slug
